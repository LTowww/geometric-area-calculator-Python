# 📐 Calculadora de Áreas em Python

Uma calculadora simples de áreas de formas geométricas desenvolvida em Python. O programa funciona via terminal e permite ao usuário escolher uma forma geométrica e calcular sua área com base nos valores informados.

## 🚀 Funcionalidades

* Cálculo da área das seguintes formas:

  * 🔺 Triângulo
  * ▭ Retângulo
  * ⚪ Círculo
  * ◼ Quadrado
* Interface simples via terminal
* Menu interativo para escolha da forma geométrica
* Código organizado com uso de funções

## 🛠️ Tecnologias Utilizadas

* Python 3
* Biblioteca padrão (`os`)

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

3. Execute o programa:

```bash
python main.py
```

## 💡 Como Funciona

Ao iniciar o programa, será exibido um menu com opções de formas geométricas. O usuário deve escolher uma opção digitando o número correspondente.

Em seguida:

* O programa solicita os dados necessários (como base, altura ou raio)
* Realiza o cálculo da área
* Exibe o resultado no terminal
